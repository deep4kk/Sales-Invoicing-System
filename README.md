# Sales Order & Invoicing System

A comprehensive Google Apps Script system for managing sales orders, invoices, and payments with inventory integration.

## Features

- **Order Management**: Create and track sales orders
- **Invoice Generation**: Auto-generate invoices from orders
- **Inventory Integration**: Auto-deduct inventory on order fulfillment
- **Payment Tracking**: Track payment status and history
- **Customer Management**: Manage customer details and credit limits

## Setup

1. Create Google Sheet with: Users, Customers, Products, SalesOrders, Invoices, AuditLog
2. Deploy as Web App
